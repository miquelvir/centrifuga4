set up 2FA

first-time set up#

  1. Install a TOTP compliant application. We recommend Microsoft Authenticator for Android or Microsoft Authenticator for iOS in your smartphone.
  2. Scan the QR code you will be shown during the sign-up process.

  2. Open the 2FA app in your smartphone.
